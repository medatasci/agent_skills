Promptable Search And Install

SkillForge separates discovery from installation, while still supporting one natural-language user request.

Resolve intent

  1. Read local generated indexes.
  2. Search exact IDs, descriptions, tags, and peer catalogs.
  3. Rank candidates by task fit and source.
  4. Ask when matches are ambiguous.

Install deterministically

  1. Verify checksum and source URL.
  2. Re-run structural validation.
  3. Copy or symlink into Codex global or project scope.
  4. Return human-readable and JSON output.

Contribute By PR

Anyone can propose a skill. The Python tool validates before upload and again before install.

python -m skillforge validate ./skills/my-skill
python -m skillforge upload ./skills/my-skill --owner medatasci
