# Improvement Loop Initiatives

Use this folder for multi-run initiative notes that are too large for a single
run log.

Good initiative notes include:

- Goal and owner.
- Why the work matters for SkillForge users and agents.
- Source URLs or local files reviewed.
- Current status: active, blocked, paused, or done.
- Files changed or planned.
- Tests, checks, and acceptance criteria.
- Open questions and next action.

Keep source-backed claims tied to URLs, local paths, commits, papers, model
cards, dataset cards, or existing SkillForge requirements.
