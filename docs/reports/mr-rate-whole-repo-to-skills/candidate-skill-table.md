## Skill Family Decision

The whole repository should be represented as a skill family, not one large
execution skill. MR-RATE contains several workflows with different inputs,
outputs, side effects, and risks:

- gated multi-terabyte dataset downloads
- raw DICOM/PACS preprocessing and defacing
- PHI-sensitive report preprocessing
- ANTs registration derivatives
- GPU-heavy contrastive model training
- checkpoint-based zero-shot inference and evaluation
