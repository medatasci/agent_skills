# MR-RATE Cross-Disease Differential Framework

This file defines the reusable comparator frame for MR-RATE disease-category research. It does not replace disease-specific clinical reasoning; it prevents the 37 source categories from being researched as isolated silos.

## Differential Ordering Principle

For each target disease, order differential diagnoses roughly by practical frequency and relevance in brain/spine MRI report interpretation, not by global epidemiology alone. Start with common or high-confusion MR-RATE categories in the same cluster, then frequent cross-cluster mimics, then uncommon but high-consequence mimics.

## High-Priority Cross-Category Contrasts

- Infarct / lacunar infarct / watershed infarct versus gliosis, encephalomalacia, demyelination, edema, hemorrhage, metastasis, glioma.
- Hemorrhage / microbleeds / cavernous malformation / subdural hematoma versus metastasis, glioma, infarct, aneurysm-related hemorrhage, calcified or susceptibility-producing lesions.
- Metastasis / glioma / meningioma / schwannoma / pituitary adenoma versus edema, demyelination, infarct, cystic lesions, and benign variants.
- Demyelination / gliosis / encephalomalacia / atrophy / ventriculomegaly versus infarct, tumor, edema, and chronic sequelae.
- Arachnoid, pineal, Rathke, choroid plexus, lipoma, mega cisterna magna, cavum septum pellucidum, empty sella, and Chiari categories versus mass lesions, hydrocephalus, atrophy, and incidental variants.
- Disc herniation / spinal stenosis / foraminal stenosis / cord compression / vertebral hemangioma versus one another in spine MRI interpretation.
