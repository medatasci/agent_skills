SkillForge Skill Catalog

Reusable Codex workflows for people who do not want to reinvent the wheel.

  • Get YouTube Media - Search YouTube, retrieve transcripts and captions, inspect caption languages, and download authorized media for research workflows.
  • Hugging Face Dataset Viewer - Inspect Hugging Face datasets with read-only Dataset Viewer API calls for splits, rows, search, filters, parquet URLs, sizes, and statistics.

  • Skill Discovery Evaluation - Improve SkillForge skill discoverability for humans, agents, local search, generated catalog pages, and publication review.
