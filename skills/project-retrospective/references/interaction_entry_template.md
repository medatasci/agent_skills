# Interaction Retrospective Entry Template

## Interaction N - Short Title

Date:
Status:

### 1. Exactly What The User Asked

> Paste the user's wording here.

### 2. What Codex Thought The User Asked

Summarize the inferred task, assumptions, and success criteria.

### 3. Summary Of What Codex Did

List the concrete actions taken, files changed, searches run, validations
performed, commits created, and artifacts produced.

### 4. Key Findings

Record the most important discoveries, constraints, risks, or useful facts.

### 5. User Response

Record the user's response after the work was delivered. If pending, write
`Pending after delivery`.

### 6. Right, Wrong, Missed

Right:

- What worked well.

Wrong or missed:

- What was incorrect, incomplete, ambiguous, late, or not yet verified.

Follow-up:

- What should happen next because of this reflection.
